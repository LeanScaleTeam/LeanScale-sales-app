# Sales Dashboards

This dashboard serves as a comprehensive tool for tracking and analyzing sales team performance metrics.

## Sales Executive Dashboard

### Key Aggregated Metrics

The dashboard displays closed deals across multiple timeframes: current month, previous month, current quarter, prior quarter, current year, and prior year. This enables quick assessment of sales performance trends.

### Goal Tracking

Users can establish and monitor monthly, quarterly, and yearly bookings targets against closed won opportunities within Salesforce, facilitating progress monitoring toward revenue objectives.

### Pipeline Overview

- **Current Unweighted Pipeline**: Shows all open opportunities in the sales pipeline
- **Current Weighted Pipeline**: Displays pipeline adjusted for deal probability, improving forecast accuracy

### Deep Dive into Reps' Pipeline

- Weighted pipeline analysis by month and individual rep, enabling detailed performance assessment
- Unweighted pipeline totals for each sales representative

### Leaderboards

- **Year-to-Date Leaderboard**: Highlights top-performing reps with quarterly breakdowns
- **Quarterly Leaders**: Identifies top performers for each quarter

### Additional Notes

This dashboard empowers informed decision-making and sales success by consolidating key aggregated metrics, objective tracking, pipeline visibility, and team performance comparisons in one unified view.
