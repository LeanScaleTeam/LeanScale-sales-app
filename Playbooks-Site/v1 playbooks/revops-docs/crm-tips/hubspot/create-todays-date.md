## Overview

Learn how to create a calculated property in HubSpot that always displays today's date. This is useful for workflows, reporting, and calculations that need to reference the current date.

## Use Cases

- Calculate days since last activity
- Determine time until close date
- Create age-based segmentation
- Build dynamic date filters

## Implementation

Create a calculated property using HubSpot's formula builder to reference the current date dynamically in your workflows and reports.
