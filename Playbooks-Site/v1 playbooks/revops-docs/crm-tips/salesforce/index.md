# Salesforce CRM Tips

This section provides Salesforce-specific tips and tutorials for RevOps teams.

## Available Salesforce Topics

The documentation covers the following Salesforce implementation areas:

- **Record Management**: Create opportunities from contacts, display record information
- **Validation & Data Quality**: Validation rules in Flow, bypass validation rules
- **Field Configuration**: Roll-up summary fields, renaming fields and objects
- **Workflow Tracking**: Close date change counter, next step fields
- **Data Organization**: Lead stages, sales stages, customer stages, proof of concept stages
- **Lead Management**: Lead source taxonomy, lead stages
- **User Interface**: Display dynamic lists, messages to end users, custom buttons
- **Admin Tools**: Data Loader tips, Salesforce Inspector Reloaded, SFDC Navigator for Lightning
- **Record Access**: Getting faster to record ID
